How to use IframeTube


IframeTube is designed to be used alongside other adblockers (e.g., uBlock Origin) to ensure all features work correctly and to maintain your privacy.

Make sure IframeTube is enabled in the popup settings. Once enabled, the embedded player with all your favorite features will appear on YouTube video pages. Enjoy! :)

How does IframeTube work


IframeTube works by placing an embedded player on the top of the native one. The embedded player is completely ad free. Do not mistake this for the Googleâ€™s version that appears in search results, which contains plenty of ads.

IframeTube then recreates all of the native features you love, such as autoplay, theater mode, next and previous video buttons, preview and more. It does this while being lightweight, completely open source and free. If you still want to learn more, check out the full documentation below.
